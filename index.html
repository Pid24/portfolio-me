Project Terbaru

Ini adalah beberapa project terbaru yang saya buat di tahun 2024

Web UMT

The UMT website is a dedicated site for the Informatics program at UMT, offering several features such as payment information, academic information, and data on lecturers and alumni of the Informatics program.

Atraindes Ticket

Atraindes Tiket is an online ticket booking mobile using the Laravel 11 framework. This mobile also has features such as booking, payment, and transactions. Additionally, it uses the Filament CMS, which is very powerful to use.

Linktree Clone

Atraindes Apothecary

Atraindes Apothecary is a drugstore mobile using the Laravel 11 framework. This mobile also has features such as multi roles, a search feature, and it utilizes Spatie and Breeze.
